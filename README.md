# 🍰 Little Baker — Baked with Love

A beautiful, fully responsive single-page website for **Little Baker**, a custom cake and dessert bakery based in Ludhiana, Punjab.

---

## ✨ Features

- **Hero Section** — Animated headline with floating badges and a morphing image shape
- **Scrolling Marquee** — Infinite-scroll banner highlighting key offerings
- **About Section** — Brand story with stat cards (years of experience, happy customers)
- **Menu Section** — 3-column grid of product cards (Custom Cakes, Cupcakes, Cookies & Brownies, Cheesecakes, Dessert Hampers, Special Boxes)
- **How It Works** — 3-step ordering process (Choose → Customize → Deliver)
- **Gallery** — Masonry-style photo grid with hover overlays
- **Reviews** — Customer testimonial cards
- **Order CTA** — WhatsApp and Instagram order buttons
- **Footer** — Brand info, quick links, specialties, and Instagram photo grid
- **Floating WhatsApp Button** — Fixed action button for quick ordering

---

## 🛠️ Tech Stack

This is a **pure HTML/CSS/JS** project — no frameworks or build tools required.

| Technology | Usage |
|---|---|
| HTML5 | Page structure & semantics |
| CSS3 | All styling, animations, and layout (Grid & Flexbox) |
| Vanilla JavaScript | Scroll-reveal animations via IntersectionObserver |
| Google Fonts | Cormorant Garamond, Dancing Script, DM Sans |
| Unsplash | Placeholder images |

## 🚀 Getting Started

No installation needed. Just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/your-username/little-baker.git

# Open in browser
open bakery.html
```

Or simply drag and drop `bakery.html` into any browser.

## 📱 Responsive Design

The site is fully responsive and adapts to all screen sizes:

- **Desktop** — Two-column hero, 3-column menu grid, 4-column gallery
- **Mobile (< 900px)** — Single-column layout, hidden hero image, stacked sections
